# Arch Linux C++ for CV

## Software

- tmux
- fish
- vim/neovim
- git
- cmake
- make
- gcc
- clang (with openmp)
- lldb

## Library

- Eigen
- OpenCV
- spdlog
- Google Test
- gflags
- glog
- Ceres Solver

## Build

```bash
docker-compose build
```

## Run

```bash
docker-compose up -d
```

## Exec

```bash
docker exec -it archlinux_cpp_for_cv /bin/bash
```

## Stop

```bash
docker-compose down
```
